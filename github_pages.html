GitHub pages.

Serwisy hostujące repozytorium takie jak min. GitHub pomagają sprawdzić kod i udostępniać go innym użytkownikom. Wystarczy że prześlemy im link do naszego kodu aby mogli go obejrzeć. GitHub daje również możliwość opublikowania naszej pracy poprzez udostępnienie master brancha danego repozytorium. Opcja w settings w zakładce GitHub pages. Dostaniemy w ten sposób adres do naszej strony na serwerze. Adres możemy przesyłać znajomym. Dzięki temu nie tylko będą mogli obejrzeć nasz kod na GitHubie ale i obejrzeć jak strona się zachowuje podczas klikania i prezentuje wizualnie.
