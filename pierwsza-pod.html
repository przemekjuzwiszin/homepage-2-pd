Strona główna Pierwsza

Pierwsza lekcja o JavaScript.

Napisałem moją pierwszą linijkę kodu w JS i utworzyłem oddzielny plik do pisania w JS. JS jest językiem programowania, dzięki któremu na stronie można tworzyć, różne szalone rzeczy. Poznałem nowe pojęcia takie jak ECMAScript, ES6, implementacja, string, feature (funkcja), zmienna, literał, primitives (typy proste). JavaScript to nie Java i prawie wszystko w JS jest obiektem.
