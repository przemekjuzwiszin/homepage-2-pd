Nowe narzędzie Git.

Dzisiaj poznałem nowe narzędzie git czyli system kontroli wersji. Konfigurowałem gita w systemie windows 10. Pracowałem z terminalem git bash, stworzyłem nowe repozytorium i dodałem commity, dzięki którym, można śledzić zmiany w tworzonym projekcie. Poznałem mapę gitk, rozszerzenie git lens oraz platformę github, na którą wypchnąłem pierwszy projekt za pomocą git bash.
